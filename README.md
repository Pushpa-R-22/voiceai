AI Voice Assistant for Accessible Communication

Overview
AI Voice Assistant for Accessible Communication is a web-based application that uses Speech Recognition and Text-to-Speech technologies to help users communicate more effectively. The system provides real-time voice interaction, accessibility support, and call simulation features.

Features
- Speech-to-Text Conversion
- Text-to-Speech Output
- Call Simulation
- Multi-Language Support
- Dark Mode
- Large Text Accessibility
- Deaf Mode
- Mute Mode
- Visually Impaired Mode
- Chat Export Functionality

Technologies Used
- HTML
- CSS
- JavaScript
- Web Speech API
- Speech Synthesis API

Future Enhancements
- AI Chatbot Integration
- Voice Translation
- User Authentication
- Cloud Storage Support
- Mobile Application Version
